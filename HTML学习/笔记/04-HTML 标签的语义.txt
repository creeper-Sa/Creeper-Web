标题标签<h1>-<h6>
    为了使网页更具有语义化，我们经常会在页面中用到标题标签，HTML提供了6个等级的网页标题
    即<h1>-<h6>
    <h1>我是一级标题</h>
    单词head的缩写
    作为标题使用，并且一句重要性递减

特点：
    1.加了标题的文字会变粗，字号变大
    2.一个标题独占一行


段落标签和换行标签
    <P>标签用于定义段落，它可以将整个网页分为若干段落

特点：
    1.文本在一个段落中会根据浏览器窗口的大小换行
    2.段落和段落之前保有空隙

    <br/>强制文字另起一行换行显示
    每次需要换行就添加一个<br/>标签，换几行加几个。
特点：
    1.<br/>是个单标符
    2.<br/>标签只是简单地开始新的一行，跟段落不一样，段落之间会插入一些垂直的间距

字体格式化标签
<font>字体化标签
    用于设置字体尺寸、字体颜色等。属性之间用空格隔开
    <font size = '大小' color = '颜色'>#########</font>
    size 属性：设置字体的大小，可以输入的值从 1 到 7，浏览器默认字体大小是 3。
    color 属性：设置字体颜色，取值：#xxxxxx 或者 colorname

文本格式化标签
    加粗--<strong></strong>      或者    <b></b>
    倾斜--<em></em>	或者	<i></i>
    删除线--<del></del>	或者	<s></s>
    下划线--<ins></ins>	或者	<u></u>


<div>和<span>标签
    无语义，他们只是一个盒子，用来装内容的
    div表示分割.分区
    span表示跨度.跨纬
特点：
    1.div一个标签独占一行
    2.span一个标签一行可以表示多个

图像标签和路径
    <img>标签用于定义HTML页面中的图像
    <img src="图像URL”>
    src是<img>标签的必须属性，他用于指定图像文件的路径和文件名
    alt表示替换文本，当图像不能显示的时候，用文字来替换图像
    title表示提示文本，鼠标放在图像上，显示文字
    width表示设置图像的宽度
    height表示设置图像的高度
    border表示设置图像的边框粗细
注意点：
    1.图像标签可以有多个属性，但是必须写在标签后面
    2.各个属性没有顺序，但是属性之间必须用空格分开
    3.属性采取键值对的格式，属性=“属性值”

实体字符与注释
实体字符:
    有些字符，像(<)这类的，对 HTML 来说是有特殊意义的，所以这些字符是不允许在文本中使用的。
    要在 HTML 中显示(<)这个字符，我们就必须使用实体字符。
    实体字符语法结构：&实体名称;
    常见的实体字符有：
    空格：&nbsp; 小于符号：&lt; 大于符号：&gt; 双引号：&quot; 版权符号：&copy;

HTML 注释
    格式：<!-- 注释内容 -->
    注释特点：浏览器运行时不显示，即不参与代码的运行，右键查看源代码可以查看到。

HTML 列表标签
<ul>和<ol>标签
    <ul> 用来定义无序列表
    <ol> 用来定义有序列表
    <li> 标签是<ul>或者<ol>标签的子标签，用来定义列表项
    无序列表：1.各个列表项之间没有顺序级别之分，是并列的
	  2.<ul></ul>中只能嵌套<li></li>，直接在<ul></ul>标签中输入其他的标签或者文字的做法是不被允许的。
	  3.<li>与</li>之间相当于一个容器，可以容纳所有元素
	  4.无序列表会带有自己的样式属性，但在实际使用时，我们会使用css来设置
        <ul>
        <li>...</li>
        ...
        </ul>

        有序列表：1.<ol></ol>中只能嵌套<li></li>，直接在<ol></ol>标签中输入其他标签或者文字的做法是不被允许的
            2.<li>与</li>之间相当于一个容器，可以容纳所有元素
                3.有序列表会带有自己的样式属性，但在实际使用时，我们会用css设置
        <ol>
        <li>...</li>
        ...
        </ol>

        自定义列表：1.<dl></dl>里面只能包含<dt>和<dd>
                2.<dt>和<dd>个数没有限制，经常是一个<dt>对应多个<dd>

超链接标签<a>
    超链接标签是在 html 页面中提供一种页面间跳转的方式。
    语法结构：<a href=”” target=””>热点文字或图片</a>
    属性解析：
    href 属性：访问目标的 url 地址
    target：定义打开页面的方式 ,常用的两种方式取值为：
    _blank 在新的选项卡中打开；
    _self 在本页面打开（默认）；

表格标签<table><tr><td>
    HTML 中一个表格由<table>标签及一个或多个<tr>加若干<td>标签组成。
    <table>是父标签，相当于表格的容器。
    border：表格边框的宽度
    width：表格的宽度
    height：表格的高度
    cellpadding：表格边缘与内容之间的空白
    cellspacing: 单元格之间的空白
    bgcolor：表格的背景颜色
    <tr>用来定义行，写在<table>标签内
    <td>用来定义列（单元格），写在<tr>标签内
    colspan：合并列
    rowspan：合并行
    align：可以用在<table>标签中，表示表格相对于整个浏览器窗口的对齐方式；也可用在<tr>或
    <td>标签中，设置单元格内容的对齐方式，常用取值：left 居左、right 居右、center 居中。
